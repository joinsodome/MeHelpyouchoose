“帮你选择”是一个选择器，不仅可解决“今天吃什么”的问题，且相对抓阄来说，这款软件在电脑上即可使用。软件无需安装！无需安装！无需安装！
软件代码只需21行，轻松简单。
首先要保证你的电脑至少为win7，装了.NET 4.6和Visual2015。
以下为使用方法：
1.解压后打开“Software\dist\me.exe”软件。
2.依次输入让你头晕的选择，每一项回车一下，完成后输入Y*。
3.软件给出自己的选择，如果满意就输入Y*，不满意就回车。
4.最后停留一秒钟后，软件关闭。
